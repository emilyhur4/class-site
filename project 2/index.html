"The ultimate messaging app for your organization - a workspace for real-time collaboration and communication, meetings, file and app sharing, and even the occasional emoji! All in one place, all in the open, all accessible to everyone."

(according to Microsoft Support)

What is Microsoft Teams?

Teams functions as a messaging app, frequently used in corporate settings for communication. It's mainly marketed as a communication platform for the workplace, therefore the target audience is mainly companies.

Content

The text is easily readable, using a clear sans serif font.

The content is everything necessary to effective communication in the context of the workplace, and uses a format that is typically universal to most communication platforms so that users are easily able to figure out the features.

There are limited media/images, which is fine for this app. Users are able to post images in channels, and also send images in the chat.

Visual Elements/UI

The app does not necessarily maintain visual hierarchy, as the experience itself is not so much chronological or ordered as it is based on the needs and tasks of the user.

Across the app, fonts, colors, and layouts remain consistent. A navigation bar lives on the left side of the interface for constant navigation availability, the colors remain to be purple. black and white, and the font doesn't change.

Interestingly, because this app is mainly used for communication, the visual elements are not exactly stimulating. If anything, they are clean and minimal so as to not confuse users and create a clean and organized look for the app.

The app works well on desktop, but is slightly limited on mobile due to the loss of screen space.

Interactivity/UX


Interestingly, there is no set homepage. The different features of the app exist equally, and so the default tab when the user opens the app is the chat tab.

As mentioned previously, navigation is pretty simple with the use of the nav bar, which includes icons and features universal to most communication platforms, and therefore intuitive.

The "teams" tab on the app allows for different teams to be created with specific users, and within those teams, different channels can be created, whether for social events or announcements.

Recommendations

Although a navigation bar is present, perhaps a home page would be useful for users. There is no chronologically implied use of the app, but a homepage may help users to navigate the app in a way that is organized and clear.

Visually, the existing interface is fine. It communicates its content clearly and presents it pleasantly.

Interestingly enough, many user complaints about Microsoft Teams lies in the functions of the features. Sometimes the notifications can get glitchy, or some other feature function.
